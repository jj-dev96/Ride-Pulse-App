# Ride-Pulse UI -> RidePulseMobile

The project has been converted to a React Native application.

## Directory Structure

- **`RidePulseMobile/`**: The new, active React Native project.
  - Run `cd RidePulseMobile` then `npx expo start`.
  
- **`Legacy_Web_Backup/`**: The original HTML/CSS/JS web application.
  - Kept for reference. You can delete this folder if you no longer need the old web code.

## Quick Start
1. Open terminal in `RidePulseMobile`.
2. Run `npx expo start`.
3. Scan QR code with Expo Go.
